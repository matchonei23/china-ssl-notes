# china-ssl-notes

## 项目简介

本项目是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面内容主要围绕 SSL/TLS 相关技术笔记、配置参考、诊断信息等，不针对任何单一域名或具体网站。所有页面均为静态 HTML 文件，可直接通过浏览器访问。

## 目录说明

```
.
├── README.md          # 本说明文件
├── index.html         # 仓库索引页面（可选）
├── pages/             # 存放所有 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── assets/            # 静态资源文件（图片、样式、脚本等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/` 目录下为各独立 HTML 页面，每个页面内容相对完整，可单独访问。
- `assets/` 目录存放页面所需的公共资源，如样式表、脚本与图片。

## 页面归档说明

本仓库中的所有页面均为独立归档，彼此之间无依赖关系。每个页面可能包含以下一种或多种内容类型：

- SSL/TLS 协议学习笔记
- 证书配置示例与说明
- 诊断或调试信息汇总
- 其他与 SSL/TLS 相关的技术文档

页面内容可能随时间更新或补充，但不会主动移除已有页面。如需引用，请使用具体文件的固定版本链接。

## 维护说明

- 本仓库由个人维护，更新频率不定。
- 欢迎通过 Issues 提交建议或报告问题，但请注意不要包含具体域名或敏感信息。
- 不接受直接推送修改至主分支的 Pull Request，如需贡献请先通过 Issue 讨论。
